# Revenue Root Cause Diagnostic Engine

Portfolio project analyzing the Olist Brazilian E-Commerce dataset to
diagnose root causes of revenue fluctuations, deployed as a Streamlit app.

## Setup
See project setup instructions (venv, requirements, git) in dev notes.

## Structure
- `data/raw/` — raw Kaggle CSVs (gitignored)
- `data/processed/` — cleaned parquet files (gitignored)
- `src/` — core Python logic
- `app/` — Streamlit app
- `notebooks/` — exploratory work (not shipped)
